# cp_crud

Simple CRUD (Create Read Update Delete) app with sqlite and flutter.

# App: Contacts

This app will allow you to:
1. Create a contact with a name and phone number
2. View all contacts
3. Edit contact by either editing their name or phone number
4. Delete a contact
